🔧 Integration Steps

Step 1: Add CSS

Add this to your HTML <head> tag:

<link rel="stylesheet" href="chatbot.css">

Step 2: Add HTML

Copy the contents of chatbot.html and paste at the end of your <body> tag.

<!-- Paste entire chatbot.html content here -->

Step 3: Add JavaScript

Add this before your closing </body> tag:

<script src="chatbot.js"></script>

Step 4: Configure API Key

Open chatbot.js and update line 23:

// Get your key from: https://makersuite.google.com/app/apikey const API_KEY = "YOUR_ACTUAL_API_KEY_HERE";
✅ That's it! Your chatbot is now ready to use. No additional setup required.

🎨 Customization Guide

Change Colors

Edit the CSS variables in chatbot.css:

:root { --primary-color: #2563EB; --primary-dark: #1E40AF; --user-message-bg: rgba(37, 99, 235, 0.15); --ai-message-bg: #F8FAFC; }

Adjust Size

Modify width and height in chatbot.css:

:root { --chatbot-width: 420px; --chatbot-height: 600px; }

Move Position

Edit .chatbot-container in chatbot.css:

.chatbot-container { bottom: 20px; /* Change these values */ right: 20px; /* to position the chatbot */ }

Modify AI Behavior

Edit the SYSTEM_PROMPT in chatbot.js to change the AI's expertise and personality.

❓ Troubleshooting

Chatbot not appearing?

• Check all 3 files are linked correctly
• Open DevTools (F12) and check Console tab for errors
• Verify HTML element IDs match JavaScript references

API errors when sending messages?

• Verify API_KEY is set correctly in chatbot.js
• Get a new key from https://makersuite.google.com/app/apikey
• Check Google API quota in your Google Cloud console

Styling looks wrong?

• Ensure chatbot.css is linked after your main CSS
• Check for CSS conflicts with your existing styles
• Clear browser cache (Ctrl+Shift+Delete)

Messages not sending?

• Check network tab in DevTools for failed requests
• Ensure you have an active internet connection
• Try refreshing the page and testing again
